**Library Management System** là một hệ thống quản lý thư viện giúp quản lý sách, danh mục, người dùng và quá trình mượn – trả sách.  
Dự án được xây dựng nhằm mục đích **học tập, thực hành Fullstack Web (Backend + Frontend)** và mô phỏng một hệ thống thư viện thực tế.
## 🛠️ Công nghệ sử dụng ( Hiện tại đây là dự án đầu tay nên sai sót mong mn thông cảm ) 

### Backend
- Java 17+
- Spring Boot
- Spring Security
- Spring Data JPA (Hibernate)
- RESTful API
- MySQL
- Swagger (OpenAPI)

### Frontend
- ReactJS
- Ant Design
- Axios
- React Router
- Vite
